MoveUs Feed design

During the creation of our application MoveUs we found the need to create a UI design that we can rely on while the app was in development. While the Figma prototype features many different screens and options the feed is one of the main parts of the application as it allows the user to search the events that have been recommended to him via the algorithm.



The feed screen features many different events that the user can select and is brought to the event information page. In the said page the user can view the details regarding the selected event, while in Figma the prototype only allows viewing of only one event as a placeholder instead of showing a different screen for every event.
The user can view participants and organizers of the event, it is also worth mentioning that the design has already undergone certain changes and ideas have been refined so the screens may differ in the final version of the product.
