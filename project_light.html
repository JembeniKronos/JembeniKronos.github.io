Light testing

During the Computer Graphics course one of our tasks was to design a model featuring metalic or objects of similar reflective surface to test out the rendering properties of light inside of Blender. In this project I decided to test it on 3 balls of different colors and roughness that allowed for different shines of light on different objects.



The models and textures are simple and are using the basics of Blender as we have only started to learn inside the program. The project contains a shining object as well as a background light that both have different effects on the reflective objects.
