MoveUs Map design

During the creation of our application MoveUs we found the need to create a UI design that we can rely on while the app was in development. While the Figma prototype features many different screens and options the map option is significat as it allows the user to view nearby events on an interactive map based on their location.



The map screen consists of the main map of the area that the user is currently in and via the pointers shows what areas feature an event soon. The user can scroll through the map as they choose as it will have an integrated map API. This option is not available in the Figma prototype as Figma offers only a certain amount of features to the user for free, however the user can select a point on the map that features an event and can see the event preview.
Said features and UI can change based on the collective opinion of the team.
